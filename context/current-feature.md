- **(SEO, low, hardening)** `APP_URL` in `src/app/sitemap.ts`, `src/app/robots.ts`, and `src/app/page.tsx` is string-concatenated with paths (`${APP_URL}/login`, `${APP_URL}/sitemap.xml`, JSON-LD `@id`s). This is correct only because `NEXT_PUBLIC_APP_URL` currently has no trailing slash.
