# Drawer Import Export

## Overview

Add import and export icon button that allow user to import and export markdown files. Showed when user open new note in drawer.

## Requirements

- Files that can be imported is markdown file and txt file only
- Export notes into markdown file, add text such like "powered by Smark" in the end of exported markdown file
- Not upload to r2, but save to our db when import
- Build related APIs, utilities with prisma query
