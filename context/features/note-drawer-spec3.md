# Note Drawer Spec

## Overview

Add pinned, favorite and delete button on drawer

## Requirements

- Muted color for outline only pinned and favorite icon. After clicking, turn to primary color with 30 opacity and color fulfill the icon
- Delete button with destructive variant. Popup alertdiaglog modal for second confirm to delete note
- Toast show after note getting pinned, favorite, delete
- Delete button locate in the middle of save and export button
- Pinned and favorite icon locate in left of note's title
